## ğŸ› Troubleshooting

### "Nothing happens when I press Ctrl+V"

**Solution:**
1. Click on the canvas (gray grid area)
2. Make sure you're not in sidebar input/textarea
3. Try clicking empty space, then Ctrl+V

### "Component appears in wrong place"

**Solution:**
- Move mouse to desired location BEFORE pressing Ctrl+V
- Component appears near mouse cursor

### "Says clipboard is empty"

**Solution:**
1. Go back to extension
2. Click the capture again
3. See "React code copied!" message
4. Then Ctrl+V on canvas

### "Code pasted but not rendering"

**Solution:**
- Check if code has syntax errors
- Look at preview area in component
- Check browser console (F12)
- Try editing code in sidebar to fix
