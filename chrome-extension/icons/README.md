# Extension Icons

Add your extension icons here:

- `icon-16.png` - 16x16px (toolbar)
- `icon-48.png` - 48x48px (extension management)
- `icon-128.png` - 128x128px (Chrome Web Store)

## Quick Generation

You can use any of these methods to create icons:

### Option 1: Online Icon Generator
1. Go to https://www.favicon-generator.org/
2. Upload your logo/design
3. Generate icons in multiple sizes
4. Download and rename to match required names

### Option 2: Figma/Photoshop
1. Create a 128x128px artboard
2. Design your icon (simple, recognizable)
3. Export as PNG in 3 sizes: 16px, 48px, 128px


## Recommended Design

- **Style**: Modern, flat design
- **Colors**: Blue gradient (matches Grab AI branding)
- **Symbol**: Target, crosshair, or cursor icon
- **Background**: Solid or gradient
- **Readability**: High contrast, simple shapes

## Temporary Icons

For testing, you can use solid color squares:
- Create PNG files with solid blue background
- Add white text "16", "48", "128" for each size
- Replace with professional icons before publishing
